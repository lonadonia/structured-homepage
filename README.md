# StructuredтДв тАФ Premium Homepage Design Prototype

A contest-ready homepage concept for **StructuredтДв** тАФ the framework for
understanding, evaluating, and applying structure within generative search
and digital information.

Concept: **"Light Core, Dark Shell"** тАФ the page opens inside the dark
intelligence layer, resolves into a light, rigorously structured core, and
closes back in the dark. The scroll itself enacts the brand promise.

## Quick start

```bash
npm install
npm run dev      # http://localhost:3000
```

Stack: Next.js (App Router) ┬╖ React ┬╖ TypeScript ┬╖ Tailwind CSS v4 ┬╖
design tokens ┬╖ hand-built SVG visuals. No backend, no external APIs,
no runtime dependencies beyond React/Next.
